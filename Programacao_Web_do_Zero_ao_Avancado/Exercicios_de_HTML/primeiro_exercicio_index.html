Introdução e História do Python


Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional e de tipagem dinâmica e forte. Ela foi concebida no final dos anos 80 por Guido van Rossum no Centro de Matemática e Informática (CWI), nos Países Baixos, como uma sucessora da linguagem ABC. O foco principal do Python sempre foi a produtividade do programador e a legibilidade do código.

O nome da linguagem não vem do réptil (a serpente píton), mas sim do grupo humorístico britânico Monty Python, do qual Guido van Rossum era um grande fã. O lançamento oficial da versão 1.0 ocorreu em 1994. Ao longo dos anos, o Python evoluiu drasticamente, com o lançamento do Python 2.0 em 2000 (que introduziu list comprehensions e garbage collection) e o Python 3.0 em 2008, que quebrou a retrocompatibilidade para corrigir falhas de design da linguagem.

Hoje, o Python é uma das linguagens mais populares do mundo, amplamente utilizada em áreas como Desenvolvimento Web, Inteligência Artificial, Ciência de Dados, Automação de Tarefas e até no desenvolvimento de scripts para videojogos. A sua filosofia central está resumida no "The Zen of Python", que defende princípios como "o bonito é melhor que o feio" e "o simples é melhor que o complexo".


Guia de Instalação do Python

Para começar a programar em Python, precisamos de instalar o interpretador oficial no sistema operativo. O processo é simples e direto em qualquer plataforma.


No Windows

1. Acede ao site oficial python.org (https://www.python.org/downloads/) e faz o download da versão estável mais recente do Python 3.
2. Executa o instalador descarregado.
3. MUITO IMPORTANTE: Na primeira janela do instalador, marca a caixa que diz "Add python.exe to PATH" (ou "Adicionar Python ao PATH"). Se não ativares esta opção, não conseguirás executar o Python a partir do terminal ou do VS Code.
4. Clica em "Install Now" e aguarda a conclusão.


No Linux (Ubuntu/Debian)

A maioria das distribuições Linux já vem com o Python pré-instalado. Podes atualizar ou instalar abrindo o terminal e executando os comandos:
sudo apt update
sudo apt install python3 python3-pip

Para garantir que o Python foi instalado corretamente, abre o teu terminal (ou Prompt de Comando) e digita o comando:
python --version (ou python3 --version). O terminal deve retornar a versão exata que instalaste.


Estruturas Essenciais da Linguagem

O Python destaca-se por não utilizar chavetas {} para definir blocos de código, ao contrário do JavaScript ou C++.
Em vez disso, ele utiliza a indentação (espaços em branco no início da linha) para organizar a estrutura do código. Se a indentação estiver errada, o código falhará.


Variáveis e Tipos de Dados

Em Python, as variáveis não precisam de uma declaração explícita de tipo. O interpretador descobre o tipo de dados automaticamente com base no valor atribuído (tipagem dinâmica).
Os tipos básicos incluem int (inteiros), float (números reais), str (texto/strings) e bool (booleanos: True ou False).


Estruturas Condicionais (if, elif, else)

As estruturas condicionais permitem que o programa tome decisões e execute caminhos diferentes no código com base em condições lógicas.
Utilizamos 'if' para a primeira condição, 'elif' (abreviação de else if) para condições intermédias e 'else' para capturar qualquer caso que não tenha sido atendido anteriormente.


Estruturas de Repetição (Loops)

Os loops são utilizados para repetir um bloco de código várias vezes. O Python possui duas estruturas principais: o 'for', que é ideal para iterar sobre uma sequência (como uma lista de elementos ou um intervalo de números), e o 'while', que continua a executar enquanto uma determinada condição for verdadeira.


Funções (def)

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Elas ajudam a evitar a repetição de código e a manter o projeto organizado.
Definimos uma função usando a palavra-chave 'def', seguida do nome da função e de parênteses.
